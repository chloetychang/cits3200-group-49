
In order to run the app, you need to set up a python virtual environment

cd into a directory outside of the project file or you can put it in the project file and add it to gitignore

you'll use this venv to store the python requirements and run the app

input these  into terminal once in the directory you want to put the venv into
python -m venv venv
source venv/bin/activate   # if you're on macOS/Linux
venv\Scripts\activate      # if you're on Windows

to get dependencies use:
pip install -r requirements.txt
